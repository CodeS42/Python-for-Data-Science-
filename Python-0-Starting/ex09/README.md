# ft_package
This is a basic Python package created as part of a project at Ecole 42.

## Description
This package contains the ‘count_in_list’ function:
- It takes a list and an element as parameters.
- It returns the number of times the element appears in the list.

## Installation
To install this package, you can use the command:
```pip install ./dist/ft_package-0.0.1.tar.gz```
or:
```pip install ./dist/ft_package-0.0.1-py3-none-any.whl```

## Usage
To use this package, simply import its functions into any Python script:
```from ft_package import count_in_list
print(count_in_list(["toto", "tata", "toto"], "toto"))```
